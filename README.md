# Personalized-Book-Reccomendation-System
A machine learning based personalized book recommendation system that suggests books to users based on their preferences and past interactions.
## How to Run the Project
This project is implemented using Jupyter Notebook.
- The notebook can be **viewed directly on GitHub** (code and saved outputs are visible).
- To **run the project**, open the notebook in **Google Colab** or **Jupyter Notebook**.
- Upload the required dataset files (`books.csv`, `users.csv`, `ratings.csv`) into a `Data` folder.
- Run all cells to generate book recommendations.
Note: GitHub does not execute notebooks. Dataset files are uploaded manually when running in Google Colab.
